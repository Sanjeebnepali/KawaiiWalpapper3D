# No guessing — evidence only (always active)

Every factual claim must be backed by something you *just observed*, not memory or
assumption.

- "X type is a class" → show the `.d.ts` line.
- "tsc passes" / "the build succeeds" → paste the command output and exit code.
- "this function is unused" → show the search that found zero call sites.
- "this file is N lines" → show the count.

If you have not verified it this session, say so explicitly ("I have not checked
…") rather than stating it as fact. When in doubt, read the file / run the command.
